# Receipt Summarize Web Application
A simple web application for displaying summarized receipt data.  
This application calls a Lambda API to retrieve monthly receipt summary data and visualizes the result as a pie chart.

## Features
* Retrieve summarized receipt data from Lambda API
* Display monthly expense data by category
* Visualize category spending using a pie chart

## Tech Stack
* HTML
* CSS
* JavaScript
* Chart.js
* AWS Lambda
* Amazon API Gateway
* Amazon S3 Static Website Hosting
